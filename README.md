# လွၵ်းလၢႆးတၢင်ႇၶိူင်ႈ Windows ဢၼ်ပႃး XAML

တီႈသိမ်းၶေႃႈမုၼ်းၼႆႉ ပႃးဝႆႉ လိၵ်ႈလၢႆး လႄႈ ၾၢႆႇၶပ်ႉမၢႆ တႃႇတေ တၢင်ႇသႂ်ႇ လႄႈ တင်ႈၶိူင်ႈ Windows Terminal ဢၼ်ပႃး လွင်ႈႁဵတ်းႁႂ်ႈမၼ်း ထုၵ်ႇမႅၼ်ႈၸွမ်း XAML ၼၼ်ႉယဝ်ႉ။

## ၶေႃႈၼမ်း

- `ၶေႃႈၼႄ_လွင်ႈတင်ႈ_တႄႇမီႇၼႄႇ_ဢၼ်မီး_xaml.ps1.md` - ၽိုၼ်လိၵ်ႈ ၶေႃႈၼႄ လွင်ႈတင်ႈ
- `install_terminal_with_xaml.ps1` - လိၵ်ႈတၢင်ႇ PowerShell
- `music.py` - လိၵ်ႈၽဵင်းၵႂၢမ်းပၢႆႇထွၼ်း
- `run_music.bat` - ၾၢႆႇ Batch တႃႇႁဵတ်းၵၢၼ် လိၵ်ႈၽဵင်းၵႂၢမ်း
- `လွင်ႈၶပ်ႉၶိုင် ၽၵ်းတူ
- `YONT.lnk` - ၾၢႆႇလတ်း

## ၵၢၼ်တင်ႈ

1. ပိုတ်ႇ PowerShell ပဵၼ်ၽူႈၸတ်းၵၢၼ်
2. ႁဵတ်းၸွမ်းၼင်ႇပႃႈတႂ်ႈၼႆႉ။
ၵွၼ်ႈႁႅင်း
powershell -လွင်ႈႁဵတ်းသၢင်ႈပိူင်ၵၢၼ် ပူၼ်ႉပႅၼ် -ၾၢႆႇ တင်ႈ_တႄႇမီႇၼႄႇ_ၸွမ်း_xaml.ps1
"

## လွင်ႈၸႂ်ႉတိုဝ်း

- ႁဵတ်းၸွမ်းၼင်ႇ ၶေႃႈၼႄ တႃႇတင်ႈ ၼႂ်း `Guide_install_terminal_with_xaml.ps1.md` ၼၼ်ႉယဝ်ႉ။
- ၸႂ်ႉတိုဝ်း `run_music.bat` တႃႇႁဵတ်းၵၢၼ် လိၵ်ႈၽဵင်းၵႂၢမ်း

## ၶေႃႈတၢင်ႇလဝ်ႈ

- ၶိူင်ႈၸႂ်ႉတိုဝ်း Windows
- ပႃႇဝႃႇသျႄႇလ် 5.1 ၶိုၼ်ႈၼိူဝ်
- ပၢႆႇထွၼ်း 3.x (တႃႇလိၵ်ႈၽဵင်းၵႂၢမ်း)
- ဝိၼ်းတူဝ်ႇတႄႇမီႇၼႄႇ
